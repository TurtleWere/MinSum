# MinSum
 Тестовое задание
## Пояснения

1. Алгоритм за один проход, тк используются 2 переменные(firstmin, secondmin) они обновляются по мере прохождения массива, сложность 0(n) - оптимально для больших массивов.
2. Обработка исключений, проверка масива на None и количество элементов.
3. Тесты включают в себя:
	1. Обычный случай.
	2. Массив, где все элементы одинаковы.
	3. Пустой массив.
	4. Массив с одним элементом.
	5. Массив на 1 млн элементов.
	6. Массив без цифр.
